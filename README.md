# SDN na Gestão Dinâmica do Fatiamento de Redes 5G para Mobilidade Urbana Sustentável

Este repositório contém o ambiente de teste (testbed) desenvolvido no âmbito da dissertação de mestrado em Engenharia de Telecomunicações e Informática na Universidade do Minho.

O objetivo principal é estudar e avaliar mecanismos de gestão dinâmica do fatiamento de redes 5G Standalone (SA), com o apoio de tecnologias como SDN, em cenários de mobilidade urbana sustentável.

## Componentes Principais

- ~~**Open5GS** - Core 5G SA~~
- **UERANSIM** - RAN 5G gNB
- ~~**UERANSIM** - RAN 5G UE~~

## Estrutura do Repositório

A estrutura do repositório será atualizada à medida que os componentes do testbed forem sendo desenvolvidos e integrados.

## Nota

Este projeto está em desenvolvimento e será atualizado progressivamente com configurações, scripts e instruções detalhadas para replicação do ambiente de testes.
